絮

又到了一年的飞絮季。前几天还只是在校园里有些飞絮，这几天在我家这边也可以看到许多的了。即使喷了很多的水，空中还是飘散着很多的小白点点。

昔我往矣，杨柳依依，今我来思，雨雪霏霏。

是这样的诗句吗？不管了，总之我会带上口罩的。来看这周的周报吧。

生活

读了《 那一天发生的超级不可思议事件 》，幻想系短篇漫画合集。画的很可爱，故事也是怪怪的，但是很有生活的感觉。

摘录

描述电脑上的生活的两个词是 BINGO 和 OOPS。你一下就做到想做的事情的时候，就是 BINGO；你无意间做了不想做的事情的时候，就是 OOPS——这种情况在 坏 的电脑系统里（其实大多数电脑系统都是这样）是很难撤销的。

网页

详解PS混合模式

这篇文章解释了 PS 各种混合模式的算法原理。以前零散地看过几个，读文章的时候确实知道那些混合模式究竟是怎么回事了——比如说 PS 的排除（Divide），其实是翻译的问题，就是指两层的图片亮度相除。而且所有混合模式都是逐像素的运算，根本没有涉及卷积之类的事情，大概只是我记混了。

不过，记不记得住另说，即使知道了这些原理，又有什么用呢？心智模型可不是靠推理建构出来的呀。想让我们先做一个亮度标准化，然后再进行数学运算，实在是有点强人所难了。大概还是熟悉一下类别就好，具体的我还是会一个一个地试吧。

不过我还是想做一些单纯亮度映射之外的东西呢……

齿

在洗澡的时候，我咬断了一截发梢，看来我的犬齿也未尝不利。这种时候，我会记起我是哺乳动物，记起这副牙铭记的野蛮，记起斗争与进食。像《来自深渊》一样，吃掉试图杀害你的生物：同时含着敬意，或者说是介于感激与庆幸的某种情感。如果我失去了一切，至少我还有尖锐的牙。可以啮咬。穿刺。撕裂。

霉

衣服有一点发霉了。我不知道为什么在北京还能出现这种情况，大概要怪罪久违的雨。

由于房间采光不好，窗户外面就是墙，所以我很少会开窗，大概只有出门的时候才会开窗通通风。大多数时候，我会把窗帘拉上，顺带还要用遮光布挡住最后一点的光线——这样子，房间里就漆黑一片了，我喜欢这样的感觉。

我可能不需要晒太阳，不过看来我的衣服还是需要的呢。

鹊

在学校里看见一只喜鹊，落在草地上，不看我一眼。 ​ ​我想起两年前见到的那只头破血流的鸟，不知道它现在还活着吗。可能已经被野猫击杀了，可能因为细菌感染死掉了，又或许根本没有挨过一年一年、凛冽的冬天。不过我会记得它昂首挺胸的样子，即使没了引以为傲的皮毛，仍然闲庭信步，不以为意……

生命是一种特权，我觉得我得敬那只鸟一杯。 （开始喝酒）

郁

这周六的演出取消了。虽说只是一场小演出，但看到通知的时候队里还是有点失落。朋友已经大四了，离毕业的时间也越来越近，他会感觉很遗憾的吧。我没有感觉很难过——我最近似乎没有多余的难过分给这件事情了。

我觉得我给自己上的压力有点太大了。压力可以变成动力，于是我给自己安排了很多事情，而我发现我撑不住了。这或许就是 Burnout 的感觉，我大概已经要燃尽了。哈哈， 半燃 ，我现在基本快烧成渣渣了。

我一直在想，我究竟要去哪里呢？我究竟要做什么呢？但我还是什么都不知道，我能知道什么呢？我有点想念乔布斯说的话了，某些地方是知道自己要做什么的人去的，而有些地方是不知道自己要做什么的人去的——我想知道，这种理想乡，究竟是哪里呢？

他说要继续读博士了，我说我不想再读书了。其实我想读书。我想学人机交互啦、学物理啦——我想学很多有趣的东西。可是我觉得自己太笨了，不太合适了。如果再聪明一点，再努力一点的话，我或许真的会去学想学的东西吧？可能那时候我又会后悔了，又会发现自己被深深地蒙骗了。

等他博士毕业的时候，我还会在吗？我会在做什么呢？我还依然热爱现在热爱的事情吗？我还有没有燃尽呢？我确实什么都不知道了。回忆过来，大学生活或许是一场盛大的失败。绩点路边一条、技术水平招笑、吉他弹得也不好，很多我想做的事情，都在脑袋里积上厚厚的灰了。我憧憬过的大学生活、憧憬过的二十岁的人生，或许从未存在过。

条条大路走不通，究竟是何意？ 我时常会失眠，到深夜，到凌晨，到曙光微明的时候。焦虑很多事情，推演很多事情，过度思考更多的事情。二十岁的、大人的世界……对我来说有点太过艰难。

试

体验了 cursor 、 firebase 、 windsurf 这三款 AI 辅助的代码编辑器。用起来果然感觉比 Visual Studio Code + Github Copilot 的组合方便一些。

我觉得或许可能还是 cursor 做的最好，至少，用起来的体验会比较顺畅。我确实感觉 AI 加速了我的编辑流程，可以自动跳转光标实在是太方便了，很多时候提示的确实是我想要做的事情。Agent 模式也很强大，虽说犯蠢的时候也不会太少。

我也深入地体验了一下 vibe coding，这确实是一种强大的方法。不过，我要说，如果你一下子没有看住，代码就很容易变成一团屎山，然后超过上下文窗口的长度。然后，就永远救不回来了。在我完全相信 AI ，做出来一坨屎之后，我决定还是自己写一个框架。至少项目结构、类、函数名这些东西还是自己写一下比较好。或者，干脆写好伪代码，然后让 AI 负责具体实现，会更符合程序员的习惯？

不过，我还是不太习惯 cursor ，我发现 VS Code + Augment Code 其实也挺好用的。

虚

答辩的时候，我向老师展示了 JSON 结构的数据，却没有想到老师的提问：这个应该怎么读呢？

我实在是不能想象，制作了那么多信息设计的老师，会不认识 JSON 格式。那些图表，究竟是怎么做出来的？难道使用 pandas 的人，连这么通用的 JSON 都不认识吗？ 我顿时有些恍惚。

我想起我很久以前在想的事情： 设计学或许是一场幻梦 。设计师除了隔离出来的设计理论，又懂得什么呢？做特定领域的设计的时候，难道没有详细的了解、没有切身的体会、依靠两三天的查资料就能做出符合要求的好设计吗？设计学或许从一开始就是虚无的事情。

设计学的缺陷和力量，都来自于设计的综合性。设计学是因工业化批量制造的需求而分工出的学科；是因理性主义手脑分离的悬浮愿景，而把计划构思从制作中切割出来的学科；是通泛而不知其所以然的「交叉学科」。 设计学的缺陷和力量，都来自于设计的综合性——我要永远记着这一点。
