后记

用 Python 写了一个简单的程序，这也是我第一个 Github 项目。

NotionTEN 是一个基于 Github action 和 Notion 的简单 Todo-list 邮件提醒服务，全称为 Notion Todo Email Notifier。

做这个项目的起因是我总是忘记作业什么时候交，就想找一个 Todo 软件。我以前试过微软的 TODO、滴答清单之类的东西，但是我觉得这些软件都太庞大了，很多功能我根本都不需要。我希望有一个跨平台的、轻量的 Todo-list 服务，于是我选择使用 Notion 与电子邮件的方案，做了 NotionTEN。

其实这个项目一开始应该是一个设计实践，我试图用五步骤法写一个设计文档，但是写着写着就不知道怎么写了，我感觉可能这个不适合用上课教的办法。好吧，其实是我已经想好怎么做了，再去回过头来写文档就有点尴尬，于是我直接把文档丢掉开始写代码（

水平只能说就那样了，写从 notion 获取数据和发邮件的函数就花了我好久（哈哈了，其实总共就两个函数），Github action 也是完全不懂。总的来说就是从上到下什么都不会，所以写这个项目我感觉还是收获了挺多的。
